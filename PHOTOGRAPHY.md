# Photography Brief — healthwelltech21

## Lighting
- North-facing window, 8–10 AM or 4–5 PM only
- White foam-board reflector opposite window
- Never flash, never overhead lighting

## Surfaces
- Unfinished oak board
- Oatmeal or rust linen
- Raw plaster wall
- Travertine tile

## Props
- Matte cream ceramic mug
- Brass spoon
- Eucalyptus stem
- Brown apothecary bottle
- Linen notebook

## Shot List (Per Post)
1. Hero flat-lay at 45°
2. In-context lifestyle shot
3. Detail macro
4. Pinterest vertical (top-third negative space for text overlay)

## Lightroom Mobile Preset
- Exposure: +0.15
- Highlights: -25
- Shadows: +20
- Whites: -10
- Temperature: +4
- Tint: +6
- Vibrance: +8
- Saturation: -12
- HSL Green: Sat +10, Lum +15
- HSL Orange: Lum +20
- Grain: 8

## Image Dimensions
- Hero: 1600×2000 (4:5 portrait)
- Post body: 1200×800 (3:2 landscape)
- Pinterest pin: 1000×1500
- OG image: 1200×630
- Thumbnails: 600×750

## File Naming
`[slug]-[shot-type]-[sequence].jpg`
Example: `sea-moss-review-hero-01.jpg`
